Task 3: SQL for Data Analysis

Objective
The objective of this task is to use SQL queries to extract (get) and analyze structured data from a dataset.

Dataset
Superstore dataset (contains sales, profit, customer and product details)

Tools Used
- MySQL Workbench

What I Did

1. Data Import
- Imported CSV dataset into MySQL database

2. SQL Queries

I performed different SQL operations to analyze the data:

- SELECT: to view required columns
- WHERE: to filter data based on conditions
- ORDER BY: to sort data
- GROUP BY: to summarize data
- HAVING: to filter aggregated results
- Subqueries: to perform nested queries
- Views: to store reusable queries

3. Analysis

- Identified high sales records
- Compared category-wise sales
- Found average and total values
- Created summary view for better analysis

Conclusion
This task helped me understand how to use SQL for real-world data analysis. I learned how to write queries to extract insights and work with structured data effectively.
